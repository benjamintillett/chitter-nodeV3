Node chat
==========

A simple chat application written in node

Skills learnt/used
-----------------
TTD - with Mocha and zombie
Nosql - with MongoDb
Node
Express
Mocha


How to use
----------

Clone the repository:

```shell 
$ git clone git@github.com:benjamintillett/node_chat.git
```


run rspec to view the tests:

```shell 
$ mocha
```
