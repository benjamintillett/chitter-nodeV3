How to use
----------

Clone the repository:

```shell 
$ git clone git@github.com:benjamintillett/node_chat.git
```


run mocha to view the tests:

```shell 
$ mocha
```

Start up the server 

```shell 
$ npm start
```

Point your browser to localhost:3000
